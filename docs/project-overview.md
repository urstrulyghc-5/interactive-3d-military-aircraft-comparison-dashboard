# Project Overview

## Interactive 3D Military Aircraft Comparison Dashboard

This project is an aerospace intelligence and benchmarking platform designed to compare some of the most influential military aircraft ever developed.

The dashboard combines operational performance metrics, strategic mission profiles, procurement economics, and capability analysis into a unified interactive experience.

### Aircraft Included

* SR-71 Blackbird
* B-2 Spirit
* B-1B Lancer
* F-22 Raptor
* F-35 Lightning II
* F-15EX Eagle II

### Objectives

* Visualize aircraft performance metrics
* Compare strategic mission capabilities
* Analyze procurement and lifecycle costs
* Demonstrate modern dashboard design techniques
* Showcase aerospace analytics through interactive visualizations

### Technologies Used

* HTML5
* CSS3
* JavaScript
* SVG Graphics
* GitHub Pages
